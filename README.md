Мод для Minecraft про вселенную соника.

## 🚀 Быстрый старт

### Требования
- **Minecraft 1.21.1**
- **Fabric Loader 0.16.9+**
- **Fabric API 0.116.12+**
- **Java 21**

### Установка
1. Скачайте последнюю версию мода из [релизов](ссылка_на_релизы)
2. Поместите `.jar` файл в папку `mods` вашего Minecraft клиента
3. Запустите игру с профилем Fabric

### Сборка из исходников
```bash
# Клонирование репозитория
git clone https://github.com/yourusername/sonicunnamedmod.git

# Переход в папку проекта
cd sonicunnamedmod

# Сборка мода
./gradlew build

# Готовый .jar файл будет в папке build/libs/
